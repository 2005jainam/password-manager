Security Tips

Don't use: common words or sequences like password, 123456, qwerty, abc123,letmein, admin, welcome, monkey, login, dragon,iloveyou, 111111, user or easy personal info.

Recommended: use a passphrase (3+ random words) or a 12+ character password with mixed case, numbers, and symbols. For high-value accounts, prefer 16+ characters.

Other tips: use unique passwords per site, store them in a password manager, enable 2-factor authentication (2FA).
